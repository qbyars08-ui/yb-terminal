ARM — Arm Holdings

Position

Cost: $200.29 × 5 sh. Current: $213.27. Gain: +$64.90. +6.48% lifetime. 1.98% of book. (2026-05-10 close)

Acquired 4/28/2026. Pivot Sleeve / working tier in the Compute & Memory layer. NOT an anchor — anchors locked at MU/NBIS/AVGO/VST/ALMU.

Thesis

ARM doesn't make chips. It licenses the architecture every other company uses to make chips. License IP up front, collect royalty per chip shipped, collect higher royalty on ARMv9 designs (~2x v8), collect higher again on Compute Subsystems (CSS) designs (~2x v9).

Royalty revenue compounds three ways simultaneously:

  • Volume: more chips shipped (every hyperscaler has at least one ARM-based program now — AWS Graviton5, Microsoft Cobalt 200, Nvidia Vera, Google Axion, Apple M-series)
  • Mix shift: v9 royalty rate is ~2x v8; CSS royalty rate is ~2x v9. v9 mix at 25% of royalties as of Q4 FY26, target 60-70% long-term
  • Per-customer expansion: ARM share of hyperscaler compute capacity approaching 50% per CFO Jason Child; CSS expanding licensee count to 21 across 12 companies

This is the IP layer underneath the chip layer. The position benefits regardless of which custom silicon program wins. Near-100% gross margin royalty annuity on the entire non-x86 stack.

Why this name (vs RISC-V, vs ARMs licensees)

  • vs RISC-V: RISC-V is the structural threat (free licensing, China political tailwind, Tenstorrent Ascalon-X performance parity with Neoverse V3). But ARM has a multi-year head start on software ecosystem maturity, CSS time-to-market (18-month advantage per company commentary), and verification + power-performance lead. Real long-term threat for greenfield workloads; legacy migration risk to ARM's installed base is low for 5+ years.
  • vs licensees building their own (ARM AGI CPU March 2026): ARM is now both a vendor AND a customer's competitor. Risk that AWS/Google/Microsoft accelerate RISC-V investment as a hedge. Counter: AGI CPU demand reportedly doubled to >$20B in 6 weeks, supply locked for first $1B.

Kill vectors (Predator checks)

1. RISC-V hyperscaler adoption — the kill vector that worries me most. Tracked quarterly by reading hyperscaler engineering blog posts + open-source contributions to RISC-V projects. Thresholds: any new hyperscaler primary CPU built on RISC-V (not auxiliary cores) = thesis at risk; 10% of hyperscaler core deployments = full-exit trigger. 2. China export ban — ARM China is 20-25% of revenue. Active overhang. A full export ban triggers immediate position re-evaluation. 3. AGI CPU channel conflict — if Q1 FY27 print shows hyperscaler CSS license churn, the channel-conflict thesis confirms and the position trims. 4. Customer concentration + multiple compression — top customers (Apple, Qualcomm, hyperscalers in aggregate) carry concentration. A single licensee renegotiation or one disappointing royalty quarter compresses the multiple fast.

Why this name vs peers

vs MU (anchor, Compute & Memory): MU sells the memory bottleneck. ARM sells the IP underneath every memory controller and every custom silicon program shipping into AI servers. Different position in the same supply chain. Both compound on the same demand curve.

vs AVGO (anchor, Compute IP + Networking): AVGO designs the chips for hyperscalers (Google TPUs, Meta MTIA). ARM licenses the cores AVGO uses inside those designs. ARM is upstream of AVGO. Owning both = full vertical exposure.

vs RISC-V exposure (no public pure-play): Tenstorrent is private. SiFive is private. China RISC-V is not investable from US. ARM is the only public way to own the non-x86 CPU IP layer.

Why now (post-IPO valuation reset): ARM IPO'd September 2023 at ~$55B. It's more than 4x'd. The valuation looks rich on backward earnings multiples, but the FY27/FY28 earnings power is going to step-function on (a) full v9 royalty mix shift, (b) CSS ASP uplift, and (c) AGI CPU revenue if it lands.

Sizing logic

ARM at 1.98% of book is intentionally small. Add as the thesis confirms quarter-over-quarter, not all-in at entry. The position is sized to grow into a 4-6% Pivot Sleeve weight if v9 mix climbs through 35-40% over the next two prints. Promotion to anchor consideration requires (a) two quarters of v9 mix step-up, (b) signature-line test pass under fresh review, (c) RISC-V kill vector still benign.

Questions to debate

1. Does the AGI CPU vertical-integration move accelerate RISC-V adoption at hyperscalers as a hedge, or does it lock ARM in deeper as the default? 2. Where does v9 royalty mix actually plateau? Company target is 60-70%. If hyperscalers move to v10 / new architectures faster than expected, does v9 ever hit the target before the next platform transition starts the cycle over? 3. The Qualcomm Nuvia precedent (October 2025 ruling clearing Qualcomm/Nuvia of breaching ARM's ALA, March 2026 countersuit pending) — does this set a ceiling on ARM's ability to capture the v9 royalty uplift? 4. Right multiple for a "growth annuity"? ARM is not a typical SaaS growth stock. Royalty business multiples (franchise restaurant operators, pharma IP licensors), chip-design-services multiples (Cadence/Synopsys), or new category?
